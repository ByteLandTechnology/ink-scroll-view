## Reporting a Vulnerability

We take the security of our software seriously. If you believe you have found a security vulnerability in `ink-scroll-view`, please report it to us as described below.

**Please do not report security vulnerabilities through public GitHub issues.**

Instead, please send an email to [github@byteland.app](mailto:github@byteland.app).

You should expect to receive a response within 24 hours. If for some reason you do not, please follow up with us to ensure we received your original message.

Please include the following information in your report:

- Type of issue (e.g. buffer overflow, SQL injection, cross-site scripting, etc.)
- Full paths of source file(s) related to the manifestation of the issue
- The location of the affected source code (tag/branch/commit or direct URL)
- Any special configuration required to reproduce the issue
- Step-by-step instructions to reproduce the issue
- Proof-of-concept or exploit code (if possible)
- Impact of the issue, including how an attacker might exploit the issue

This information will help us to triage your report more quickly.
